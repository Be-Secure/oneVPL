Detailed Description

Item of the layer.

We parse name of the value to extrac parents structures and verify correctness of the parameter. Let's imagine we have parameter: "mfxImplDescription.Impl". Here we have two layers: "mfxImplDescription" with ID = 1 and "Impl" with ID = 2. "Impl" has parentID = 1. Finally we will have two instances of the structure.
